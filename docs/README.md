# Sistema do ECC

Sistema web para gerenciamento de Encontro de Casais com Cristo (ECC).

## Tecnologias

- Python 3
- Flask
- MySQL
- Jinja2

## Estrutura

- routes/ → rotas da aplicação
- services/ → regras de negócio e acesso ao banco
- templates/ → páginas HTML
- static/ → CSS e JS
- db.py → conexão com banco (pool)
